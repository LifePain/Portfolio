02 · About

A solo dev, not a solo hobbyist.

I write design docs before I write code. Every system I ship has a spec, a test plan, and a reason to exist.

Focus

Simulation-driven gameplay. Large-scale AI orchestration. Emergent narrative systems. Procedural worlds you can replay.

Stack

Unity 6 · C# · DOTS/ECS · Netcode for Entities · Unity.Mathematics · Burst. Comfortable with shader work, procedural generation, and network optimization for thousands of units.

Method

Spec-first design. Vertical slices. Test-driven core systems. I ship one feature module at a time and make sure it holds before moving on.

Project 02

A tower. A cast of heroes. Permadeath.

Pick Me Up is a tower-climbing manager. You summon heroes with real personalities, train them in facilities you build, and send them into a procedurally generated tower. Every death is permanent. Every world is unique.

Role
Solo Dev
Genre
Tower-Climb Management
Engine
Unity · 2D Pixel Art
Status
Design Complete · Building Core
  • AI personality system. Heroes remember, form bonds, refuse orders.
  • Seed-based deterministic worlds. Every account is unique.
  • Discovery-based skill fusion with narrative backlash.
